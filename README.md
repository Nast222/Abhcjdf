Book TrackerАвтор: [Фирсова]

ОписаниеПриложение для учёта прочитанных книг с возможностью добавления новых книг через форму (название, автор, жанр, количество страниц), фильтрации по жанру и количеству страниц. Все данные сохраняются в файл books.json и загружаются при запуске.

Инструкция по запуску

Установите Python 3.6+.
Скачайте или клонируйте репозиторий.
Запустите: python book_tracker.py
Используйте интерфейс для работы с книгами.
Примеры использования

Введите данные о книге и нажмите «Добавить книгу».
Используйте фильтры для поиска по жанру или количеству страниц.
Данные сохраняются автоматически.
